This is a simple text comparator. You can analyze up to 3 texts. To generate the .html presentation you need to:
- install R and Rstudio
- put texts in the directory texts and rename them as text_1.txt, text_2.txt, text_3.txt
- copy and paste the path of your copy of the file TextAnalysis in generate.R
- open terminal and write "Rscript" the write he path of your generate.R or simply drag and drop if your use macOS
